# 分享一个公众号批量上传图片获取图片链接的脚本.otl

分享一个公众号批量上传图片获取图片链接的脚本  
### 脚本说明：

这个脚本是我自己写公众号时生成整体文章的时候配合用的。  
有时候我们写文章会有很多配图，一张一张上传再放到对应位置很麻烦；  
我的文章又是用markdown语法写的，有了链接后直接替换文本里的图片链接就好了。  
  
比如这样一份基于markdown语法写的文章：  

```markdown
## 标题

正文内容
图片1
正文内容
图片2
正文内容
图片3
正文内容
图片4
正文内容
图片5
```

只需要批量上传好图片获得链接，把里面相应位置的链接替换一下就好了。  
上传图片获得链接之后还可以用于其他地方引用，最近我的编辑需要整一份资源型的文章，里面大量配图，我就干脆单独整一个脚本来上传，顺带分享给大家。  
  
### 启用权限：

想要公众号有使用上传接口的权限，需要在公众号后台启用一些配置：  
![image]()
  
上图是开启后的，如果没有开启，按照操作指引即可：  
![image]()
需要扫码，验证密码，然后启用即可，这样我们就获得了api接口的AppID和AppSecret，类似账号密码。  
注意：AppSecret启用的时候复制下来，记得保存，公众号后续不会再显示，忘记了只能回去基本配置那里重置。  
  
下面是完整的脚本代码：  

```python
from requests import post as re_post
from requests import get as re_get
from pickle import load,dump
from os import path,listdir
from time import time

# -----------------------------------
# 目标图片所在文件夹
dir_path = r'换成你的目标图片所在文件夹路径'
# 公众号API账号密码
user_pass = {'appid':'换成你的id','secret':'换成你的密码'}
# -----------------------------------

# 更新Token
def update_token(user_pass):
    api = 'https://api.weixin.qq.com/cgi-bin/token'
    params = {
        'grant_type':'client_credential',
        'appid':user_pass['appid'],
        'secret':user_pass['secret']
    }
    rel = re_get(api,params=params)
    if 'access_token' in rel.json():
        with open('token_recopt.pk','wb') as file:
            dump({'access_token':rel.json()['access_token'],'expires_in':time()},file)
        return 0,rel.json()['access_token']
    return 1,rel.json()
# 上传图片
def upload_img(access_token,img_path):
    api = 'https://api.weixin.qq.com/cgi-bin/media/uploadimg'
    params = {'access_token':access_token}
    img_file = {'media':open(img_path,'rb')}
    rel = re_post(api,params=params,files=img_file)
    return rel

# 检查token时效
token_recopt = {'access_token':'','expires_in':0}
if path.exists('token_recopt.pk'):
    with open('token_recopt.pk','rb') as file:
        token_recopt = load(file)
if (time() - token_recopt['expires_in']) > 7000:
    code,access_token = update_token(user_pass)
    if code == 0:
        token_recopt['access_token'] = access_token
    else:
        print(access_token)
        exit()

for file_name in listdir(dir_path):
    if not file_name.lower().endswith(('.jpg','.jpeg','.png')):continue
    img_path = path.join(dir_path,file_name)
    rel = upload_img(token_recopt['access_token'],img_path)
    url = ''
    if 'url' in rel.json():
        url = rel.json()['url']
    print('%s\t%s' % (file_name,url))
```
